Connect Four Game AI
This project is an implementation of the classic Connect Four game with two modes:
Player vs Player
Player vs AI
The AI uses game tree search algorithms such as Minimax (and Alpha-Beta Pruning, if implemented) to make intelligent moves. The project demonstrates concepts of Artificial Intelligence, game theory, and decision-making algorithms.
